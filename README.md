# My Portfolio

A stunning, modern portfolio website showcasing my projects, skills, and experience as a Full Stack Developer.

## 🎨 Features

- **Modern Dark Theme** - Sleek and professional dark design with cyan and blue accents
- **Smooth Animations** - GSAP animations for engaging interactions and scroll effects
- **Fully Responsive** - Optimized for desktop, tablet, and mobile devices
- **Project Showcase** - Beautifully displayed project cards with hover effects
- **Skills Section** - Organized display of technical skills and expertise
- **Contact Section** - Easy ways to connect via GitHub, Email, and LinkedIn
- **Performance Optimized** - Fast loading with efficient animations
- **Accessibility** - Proper semantic HTML and keyboard navigation support

## 🚀 Technologies Used

- **HTML5** - Semantic markup
- **Tailwind CSS** - Utility-first CSS framework
- **GSAP** - Advanced animations library
- **JavaScript** - Interactive features and animations
- **Responsive Design** - Mobile-first approach

## 🎬 Animations Included

- **Blob Animation** - Floating animated background shapes
- **Gradient Text** - Animated gradient text effect
- **Card Hover Effects** - Project and skill cards animate on hover
- **Scroll Triggers** - Elements animate as they come into view
- **Counter Animation** - Stats counter animates when scrolled into view
- **Smooth Transitions** - All interactions have smooth CSS transitions
- **Mobile Menu** - Smooth slide-in animation for mobile navigation

## 📱 Responsive Breakpoints

- **Mobile** - Optimized for devices under 640px
- **Tablet** - Enhanced for 640px - 1024px
- **Desktop** - Full experience for 1024px and above

## 🔧 Customization

To customize this portfolio:

1. Update your name and title in the hero section
2. Modify project information in the Projects section
3. Add your GitHub, email, and social links in the Contact section
4. Adjust colors by modifying Tailwind classes (cyan, blue, purple theme)
5. Update skills in the Skills section
